# 体脂计算器APP图标设计指南

## 图标尺寸要求

根据创建的mipmap目录结构，需要以下尺寸的图标：

- **mipmap-mdpi**: 48x48px
- **mipmap-hdpi**: 72x72px  
- **mipmap-xhdpi**: 96x96px
- **mipmap-xxhdpi**: 144x144px
- **mipmap-xxxhdpi**: 192x192px

## 推荐设计概念

### 主题设计建议：
1. **主要元素**: 可以包含身体轮廓、心形或健康符号
2. **颜色方案**: 建议使用健康主题的颜色，如蓝色、绿色或橙色
3. **风格**: 简洁的扁平化设计，符合现代Android设计规范

### 具体图标概念：

**概念1 - 身体测量主题**
- 外形: 简洁的人体轮廓
- 中心: 可显示百分比符号(%)或体重秤图标
- 颜色: 蓝色主题 (#2196F3)

**概念2 - 健康盾牌主题** 
- 外形: 盾牌形状背景
- 中心: 心脏图案 + 百分比符号
- 颜色: 绿色主题 (#4CAF50)

**概念3 - 简洁数字主题**
- 外形: 圆角矩形背景
- 中心: 大字体显示 "BF" 或 "%" 符号
- 颜色: 橙色渐变主题 (#FF9800)

## 文件命名规范

所有图标文件请命名为：`ic_launcher.png`

## 图标制作工具推荐

1. **在线工具**:
   - App Icon Generator (appicon.co)
   - Android Asset Studio (romannurik.github.io/AndroidAssetStudio)

2. **专业工具**:
   - Adobe Illustrator/Photoshop
   - Figma
   - Sketch

## 下一步操作

1. 选择一个设计概念
2. 使用推荐工具生成所需尺寸的图标文件
3. 将图标文件放入对应的mipmap目录中
4. 更新AndroidManifest.xml添加图标引用
