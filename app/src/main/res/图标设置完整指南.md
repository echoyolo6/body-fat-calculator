# APP图标设置完整指南

## 当前状态

✅ 已完成的设置：
- 创建了完整的mipmap目录结构
- 更新了AndroidManifest.xml添加图标引用
- 图标路径：`@mipmap/ic_launcher`

## 需要执行的步骤

### 步骤1: 生成图标文件

#### 方法A: 使用Android Studio自动生成（推荐）

1. 打开Android Studio项目
2. 在res目录上右键点击
3. 选择 "New" → "Image Asset"
4. 选择 "Launcher Icons (Adaptive and Legacy)"
5. 资源配置设置：
   - Icon Type: Launcher Icons
   - Name: ic_launcher
   - Source Asset: 
     - Asset Type: Clip Art 或 Image
     - 选择合适的图标（推荐搜索 "fitness", "health", "calculator" 相关图标）
   - 设置背景颜色

#### 方法B: 手动制作图标

如果你想要自定义图标：

1. 访问 https://romannurik.github.io/AndroidAssetStudio/
2. 选择 "Launcher icons"
3. 上传你设计的图标或选择提供的图标
4. 下载生成的ZIP文件
5. 解压后各个mipmap目录中的ic_launcher.png文件

### 步骤2: 图标尺寸要求

确保生成的文件尺寸如下：
- mdpi: 48x48px
- hdpi: 72x72px  
- xhdpi: 96x96px
- xxhdpi: 144x144px
- xxxhdpi: 192x192px

### 步骤3: 放置图标文件

将生成的`ic_launcher.png`文件复制到对应目录：

```bash
# 复制到项目目录
cp ic_launcher.png /path/to/your/project/app/src/main/res/mipmap-mdpi/
cp ic_launcher.png /path/to/your/project/app/src/main/res/mipmap-hdpi/
cp ic_launcher.png /path/to/your/project/app/src/main/res/mipmap-xhdpi/
cp ic_launcher.png /path/to/your/project/app/src/main/res/mipmap-xxhdpi/
cp ic_launcher.png /path/to/your/project/app/src/main/res/mipmap-xxxhdpi/
```

## 推荐图标设计

### 对于体脂计算器应用，推荐：

1. **健康主题图标**: 在Google Material Icons中搜索 "fitness_center" 或 "health_and_safety"
2. **测量主题**: 搜索 "straighten" 或 "monitor_weight"
3. **计算主题**: 搜索 "calculate" 或 "percent"

### 颜色建议:
- 主色: #2196F3 (蓝色 - 信任、健康)
- 辅色: #4CAF50 (绿色 - 健康、自然)  
- 强调色: #FF9800 (橙色 - 活力、温暖)

## 快速解决方案

如果你需要立即测试，可以临时使用任何48x48px的PNG图标文件作为占位符。

## 验证设置

1. 构建应用：`./gradlew build`
2. 在Android模拟器或设备上运行
3. 检查应用图标是否正确显示

## 故障排除

如果图标不显示：
1. 检查文件名是否为 `ic_launcher.png`
2. 确保所有mipmap目录都有对应文件
3. 确认AndroidManifest.xml中的引用路径正确
4. 清理并重新构建项目：`./gradlew clean build`
